# 语音交流平台项目总结

## 🎯 项目概述

语音交流平台是一个基于 Web 的智能语音交互系统，专注于中国传统文化的知识传播和交流。项目集成了语音识别、语音合成、知识库管理等功能，为用户提供沉浸式的文化学习体验。

## ✨ 核心功能

### 🎤 语音交互

- **语音识别**：实时语音转文字，支持中文识别
- **语音合成**：文字转语音，自然流畅的语音输出
- **智能对话**：基于知识库的智能问答系统
- **语音控制**：通过语音指令控制界面功能

### 📚 知识库管理

- **分类浏览**：传统美食、工艺、戏曲、节日、医药、建筑六大分类
- **资源导入**：支持 JSON 格式的资源文件导入
- **搜索功能**：基于关键词和标签的智能搜索
- **资源统计**：实时显示资源数量和分类统计

### 🎨 用户界面

- **响应式设计**：适配各种屏幕尺寸
- **深色模式**：支持系统主题切换
- **动画效果**：流畅的交互动画
- **资源管理**：直观的资源管理界面

### 🔧 系统功能

- **历史记录**：对话历史保存和查看
- **设置面板**：语音、界面、系统设置
- **快捷功能**：清空对话、导出记录、备份数据
- **系统监控**：实时系统状态监控

## 🚀 技术架构

### 前端技术

- **HTML5**：语义化标记
- **CSS3**：现代样式和动画
- **JavaScript ES6+**：模块化开发
- **Web Speech API**：语音识别和合成
- **LocalStorage**：本地数据存储

### 后端技术

- **Node.js**：服务器运行环境
- **Express.js**：Web 应用框架
- **Winston**：日志管理
- **Helmet**：安全中间件
- **Compression**：响应压缩

### 部署技术

- **Docker**：容器化部署
- **Nginx**：反向代理和负载均衡
- **PM2**：进程管理（可选）

## 🚀 部署方式

### 1. 传统部署

```bash
npm ci --only=production
NODE_ENV=production node server-optimized.js
```

### 2. Docker 部署

```bash
docker build -t voice-chat-platform .
docker run -d -p 25812:25812 voice-chat-platform
```

### 3. Docker Compose 部署

```bash
docker-compose up -d
```

### 4. 一键部署脚本

```bash
# Linux/macOS
./deploy.sh production

# Windows
deploy.bat production
```
