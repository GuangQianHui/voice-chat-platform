# 语音交流平台

一个基于 Web 的智能语音交流平台，支持语音识别、语音合成和传统文化知识库。

## 功能特性

### 核心功能

- 🎤 实时语音识别和语音合成
- 💬 智能对话系统
- 📚 传统文化知识库
- 📝 对话历史记录管理
- 🎨 美观的中国风界面设计

### 界面功能

- 📱 响应式设计，支持多种设备
- 🔄 侧边栏折叠/展开功能
- ⌨️ 键盘快捷键支持
- 🔔 实时通知系统
- 🎯 平滑动画效果

## 侧边栏折叠功能

### 使用方法

1. **鼠标操作**：点击侧边栏边缘的折叠按钮
2. **键盘快捷键**：
   - `Ctrl/Cmd + 1`：切换左侧对话历史
   - `Ctrl/Cmd + 2`：切换右侧知识库

### 功能特点

- 🎨 美观的渐变按钮设计
- 🔄 平滑的折叠/展开动画
- 💡 智能工具提示
- 📱 响应式适配
- 🎯 精确的点击反馈
- 🔔 操作状态通知

### 按钮位置

- **左侧对话历史**：按钮位于右侧边缘
- **右侧知识库**：按钮位于左侧边缘

## 技术栈

- **前端**：HTML5, CSS3, JavaScript (ES6+)
- **样式**：Tailwind CSS
- **图标**：Font Awesome
- **语音**：Web Speech API
- **动画**：CSS3 Transitions & Animations

## 文件结构

```
语音交流平台/
├── index.html              # 主页面
├── css/

│   └── tailwind-fallback.css
├── js/

│   ├── audio/

│   ├── managers/           # 管理器模块
```

## 快速开始

1. 克隆或下载项目文件
2. 使用本地服务器运行项目（推荐使用 Live Server）
3. 在浏览器中打开 `index.html`
4. 开始使用语音交流功能

## 浏览器兼容性

- Chrome 66+
- Firefox 60+
- Safari 12+
- Edge 79+

## 注意事项

- 需要 HTTPS 环境或 localhost 才能使用语音功能
- 首次使用需要授权麦克风权限
- 建议使用现代浏览器以获得最佳体验
