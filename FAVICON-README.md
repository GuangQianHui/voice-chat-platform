# 网站图标设置说明

## 概述

为您的语音交流平台网站添加了专业的网站图标（favicon），包含多种格式以支持不同的浏览器和设备。

## 图标文件

- `favicon.svg` - SVG 格式图标，支持动画效果
- `favicon.png` - PNG 格式图标（需要生成）
- `favicon.ico` - ICO 格式图标（需要转换）
- `generate-favicon.html` - 图标生成工具

## 图标设计

图标设计包含以下元素：

- 红色圆形背景（#dc2626）
- 白色语音波形动画
- 对话气泡图案
- 符合语音交流平台的主题

## 使用方法

### 方法一：使用图标生成工具（推荐）

1. 在浏览器中打开 `generate-favicon.html`
2. 点击下载按钮获取不同尺寸的 PNG 图标
3. 将下载的 32x32 图标重命名为 `favicon.png`
4. 将文件放在网站根目录下
5. 刷新浏览器页面查看效果

### 方法二：手动设置

1. 使用图像编辑软件（如 GIMP、Photoshop）打开 `favicon.svg`
2. 导出为 PNG 格式，尺寸为 32x32 像素
3. 保存为 `favicon.png`
4. 将文件放在网站根目录下

### 方法三：在线转换

1. 访问在线图标转换工具（如 favicon.io）
2. 上传 `favicon.svg` 文件
3. 下载生成的多种格式图标
4. 将文件放在网站根目录下

## 浏览器支持

- **现代浏览器**：支持 SVG 格式，显示动画效果
- **旧版浏览器**：使用 PNG 格式作为备选
- **移动设备**：使用 PNG 格式，支持 Apple Touch Icon

```
├── index.html           # 已更新，包含图标链接
└── FAVICON-README.md    # 本说明文件
```

## 注意事项

1. 确保图标文件放在网站根目录下
2. 清除浏览器缓存以查看新图标
3. 不同浏览器可能需要不同的图标格式
4. 建议同时提供多种格式以确保兼容性

## 自定义

如需修改图标设计：

1. 编辑 `favicon.svg` 文件
2. 使用图标生成工具重新生成 PNG 格式
3. 更新相关文件

## 测试

设置完成后，您应该能在以下位置看到新图标：

- 浏览器标签页
- 书签栏
- 历史记录
- 移动设备主屏幕（如果添加到主屏幕）
