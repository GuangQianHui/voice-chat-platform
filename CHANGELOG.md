# 更新日志

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

## [2.0.1] - 2024-08-14

### 新增

- ✨ 添加资源管理按钮到导航条
- ✨ 支持外部资源管理页面链接
- ✨ 新增资源导入功能
- ✨ 支持 JSON 格式资源文件导入
- ✨ 添加资源分类管理功能

### 优化

- 🎨 优化导航条样式和响应式设计
- 🎨 隐藏资源 ID 显示，提升界面简洁性
- 🔧 改进资源管理器导入逻辑
- 🔧 优化文件上传和验证机制
- 📱 增强移动端适配

### 修复

- 🐛 修复资源 ID 显示问题
- 🐛 修复导入功能中的字段验证错误
- 🐛 修复深色模式下的样式问题

## [2.0.0] - 2024-08-10

### 新增

- 🎉 全新语音交流平台发布
- 🎤 实时语音识别功能
- 🔊 自然语音合成功能
- 🤖 AI 智能对话系统
- 📚 传统文化知识库
- 🎨 现代化用户界面
- 📱 响应式设计
- 🌙 深色模式支持

### 功能特性

- 语音识别和语音合成
- 智能对话和上下文理解
- 知识库分类浏览
- 对话历史记录
- 系统状态监控
- 数据本地存储

### 技术栈

- 前端：HTML5, CSS3, JavaScript ES6+
- 语音：Web Speech API
- 样式：自定义 CSS + Tailwind CSS
- 图标：Font Awesome
- 服务器：Node.js + Express
